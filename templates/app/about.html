About Out Project

To better understand wildlife ecosystems, it is necessary to have detailed, large-scale knowledge about the animals in the natural ecosystems. We need to be able to accurately identify the species of animal in different places and times. Images are obtained from cameras placed in the ecosystem. Using image processing and machine learning techniques, we can identify animals based on the patterns, such as spots and stripes, found on their skin.
Initially we aim to identify tigers from a set of images. After training and testing the model on a tiger dataset, we wish to apply the same model on datasets containing other animal species.
The data then obtained can be used to learn animal movement patterns and location. This can ensure that infrastructural development happens only in areas where the animal corridors are not disturbed. An extension of this can be used to identify biometrics of individual animals, which can be used to count population and to keep track of animals.
